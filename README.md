# 基于机器学习的工业设备故障预测系统

## 项目简介

本项目面向工业设备预测性维护场景，基于设备运行传感器数据构建设备故障预测模型。

系统支持设备运行数据 CSV 读取、数据预处理、模型训练、模型评估、特征重要性分析，并支持手动输入单条设备运行参数，预测设备是否存在故障风险，同时输出故障概率、风险等级和维护建议。

## 项目背景

在工业生产场景中，设备故障可能导致停机、产能损失和维护成本增加。预测性维护的目标是在设备真正发生故障前，根据传感器数据提前识别风险，从而安排检修计划，降低突发故障概率。

本项目使用机器学习方法，根据设备运行参数预测设备是否可能发生故障。

## 数据集

本项目使用 AI4I 2020 Predictive Maintenance Dataset。

数据集字段包括：

- `Air temperature [K]`：空气温度
- `Process temperature [K]`：工艺温度
- `Rotational speed [rpm]`：转速
- `Torque [Nm]`：扭矩
- `Tool wear [min]`：工具磨损时间
- `Type`：产品类型
- `Machine failure`：是否发生设备故障

其中 `Machine failure` 为目标字段：

- `0`：设备正常
- `1`：设备故障

> 注意：本仓库不直接上传数据集文件。请自行下载 `ai4i2020.csv`，并放入 `data/` 文件夹。

## 核心功能

- CSV 数据读取与预览
- 故障样本分布展示
- 数据清洗与特征处理
- `Type` 类别字段 One-Hot 编码
- 随机森林模型训练
- Accuracy、Precision、Recall、F1-score 模型评估
- 混淆矩阵展示
- 特征重要性分析
- 模型保存与加载
- 单条设备故障风险预测
- 故障概率、风险等级、维护建议输出
